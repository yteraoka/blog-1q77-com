Cinnamon で使いそうなキーボードショートカットの一覧 Super は Windows キー

ワークスペース一覧

Ctrl + Alt + Up

Windowの最大化

Super + Up

Windowの最大化解除

Super + Down

Windowを閉じる

Alt + F4

デスクトップを表示

Ctrl + Alt + d

Window切り替え

Alt + Tab

右のワークスペースに切り替え

Ctrl + Alt + Right

左のワークスペースに切り替え

Ctrl + Alt + Left

カスタムショートカット  
Terminal起動

Ctrl + Alt + t
